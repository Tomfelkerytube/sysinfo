## How to install via apt
Run
```
echo "deb [trusted=yes] https://tomfelkerytube.github.io/sysinfo ./" | sudo tee /etc/apt/sources.list.d/sysinfo.list
```
and the run
```
sudo apt update
```
and then run
```
sudo apt install sysinfo
```
